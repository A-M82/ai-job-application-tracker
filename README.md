# AI Job Application Tracker

## Overview

AI Job Application Tracker is a full-stack web application that helps job seekers manage their applications in one place and gain valuable insights into their job search process.

Users can track application status, manage recruiter information, prepare for interviews using AI-generated questions, and analyze their job search performance through dashboards and analytics.

## Features

### Application Management

* Create and manage job applications
* Store company, position, location, salary, and job description
* Store recruiter contact information
* Upload CV and cover letter versions

### Status Tracking

* Applied
* Interview Scheduled
* Interview Completed
* Rejected
* Offer Received
* Accepted

### Dashboard

* Overview of all applications
* Upcoming interviews
* Recent activity
* Application timeline

### AI Interview Preparation

* Generate interview questions from job descriptions
* Technical interview questions
* Behavioral interview questions
* Interview preparation tips

### Analytics & Insights

* Track application success rate
* Track interview-to-offer ratio
* Visualize applications by company, industry, and location
* Identify common rejection patterns
* AI-generated suggestions for improvement based on historical application data
